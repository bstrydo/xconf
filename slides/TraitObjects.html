public interface MyInterface {
    String getName();
}

public void myMethod(MyInterface myVar) {
    System.out.println("my_var.getName() = " + myVar.getName() );
}
Usually we use Traits with generics like
fn my_func <Type : Trait>(my_var : Type)...

This means my_func can work on type Type provided it implements trait Trait.

Collections such as vectors and arrays can only hold a single type.
let my_vec : Vec<String>;

Trait objects give us the same functionality as the java code above. 

fn my_func(my_var : Box<MyTrait>).. 
let my_struct_instance = MyStruct{...};
let my_trait_object = Box::new(my_structinstance);
my_func(my_trait_object);
let my_vec : Vec<Box<MyTrait>>;
